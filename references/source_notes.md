# Source Notes

## Original visualization

Reuters Graphics. "Who is the biggest polluter of carbon dioxide?" Published Nov. 10, 2021.

URL: https://www.reuters.com/graphics/CLIMATE-UN/EMISSIONS/jnvwexaryvw/

Use this as the original published news-media visualization being critiqued and redesigned.

## Main working dataset

Our World in Data. CO2 and Greenhouse Gas Emissions dataset.

Current CSV: https://owid-public.owid.io/data/co2/owid-co2-data.csv

Repository and codebook: https://github.com/owid/co2-data

Use this as the raw dataset for the reproducible QMD pipeline.

## Reproducibility note

The Reuters article cites Our World in Data as its source. The current OWID file may differ from the 2021 Reuters values because OWID updates the dataset over time. If the aim is to reproduce Reuters' 2020 rankings closely, use the OWID GitHub commit from Nov. 5, 2021:

https://raw.githubusercontent.com/owid/co2-data/6f271c386389d9df2984fb58d8376b8bdcca5009/owid-co2-data.csv

For the redesigned poster, using the current OWID dataset is acceptable if the poster clearly states the dataset version or download date.
